Perguntas Frequentes (FAQ)

📱 O aplicativo já está disponível?

Nosso app estará disponível em breve para Android e iOS. Fique atento ao nosso site e redes sociais.

💳 Como funciona o cartão virtual?

Você pode gerar cartões virtuais diretamente pelo aplicativo, ideais para compras online com segurança.

💸 O que é saldo remunerado (Float)?

Seu saldo em conta pode render automaticamente conforme a média diária. Consulte a política de remuneração no app.

🔐 O acesso via API é seguro?

Sim. Toda comunicação com nossa API é criptografada via HTTPS e exige autenticação por token.

📨 Como posso entrar em contato com suporte?

Envie um e-mail para suporte@sbank.io ou acesse o formulário em nossa página de contato.
