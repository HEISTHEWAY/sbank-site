Documentação da API – S BANK

Essa página oferece uma visão geral das principais rotas e autenticação da API da plataforma.

Autenticação

Use o header Authorization: Bearer <seu_token> em todas as requisições protegidas.

Exemplo de requisição Pix

POST /api/v1/pix/enviar
{
  "cpf": "00000000191",
  "valor": "10.00",
  "descricao": "Pagamento teste"
}

Outros endpoints

  • GET /api/v1/conta/saldo – Consultar saldo
  • GET /api/v1/conta/extrato – Extrato de movimentações
  • POST /api/v1/cartao/emitir – Emitir cartão virtual
